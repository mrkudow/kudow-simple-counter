Fichier .TXT
Créer un fichier vide au format .txt dans un emplacement de l'ordinateur.
Par exemple, counter.txt
Utiliser le compteur sur l'application
Cliquer le bouton Choisir le fichier texte
Modifier le nom du compteur à sa guise dans le champ de texte dédié
Incrémenter ou décrémenter le score avec, respectivement, les boutons + et -
Charger le fichier sur OBS
Dans OBS, créer une source Texte (GDI+)
Cocher la case Lire à partir d'un fichier
Cliquer le bouton Parcourir puis sélectionner le fichier TXT créé
puis valider la source L'affichage se met à jour en temps réel
